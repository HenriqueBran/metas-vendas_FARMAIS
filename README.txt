SITE CALCULADORA DE METAS E VENDAS

Como usar:
1. Abra o arquivo index.html no navegador.
2. Use a tela "Metas do mês" para configurar meta, dias de trabalho, premiação e funcionários.
3. Use a tela "Lançamentos diários" para preencher as vendas por dia.
4. Use a tela "Resultados" para ver o resumo geral e por funcionário.

Observações:
- Os dados ficam salvos automaticamente no navegador usando localStorage.
- Você pode exportar/importar um arquivo JSON com os dados.
- A planilha original tinha percentuais de 15% para 7 funcionários, totalizando 105%. O site mostra alerta quando a soma passa ou fica abaixo de 100%.
